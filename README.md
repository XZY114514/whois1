# whois1
用来生成“遇到困难请拨打：”类型的抽象小图片，很好地解决了制作110格式图片需要打开图片编辑软件的缺点。
[B站教程](https://www.bilibili.com/video/BV1GUfJBtEB4/)

我们有三个格式的文件，`.html` ,`.py` ,`.exe` ,其中 `.exe` 格式的文件是个压缩包，你需要解压之后才能使用。
## 怎么选择
### 建议优先选择 `.html` 格式文件 ~~除非你闲的没事~~
### 1. Windows用户
都能选，但是 `.py` 文件需要你自己配置一个python环境编译
### 2. Mac & Linux用户
除了 `.exe` 大概都能用（我没有这两个设备没试，但是 `.py` 应该是能编译的）
### 3. Android & iOS用户
似乎只有 `.html` 能用，但是Android似乎可以自己编译python和运行exe

Tips：所有代码都是AI生成的，exe是我自己打包的（感谢Qwen3-Coder）
